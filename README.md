# Time Tracker AI MCP

Track work hours, calculate durations, and summarize time entries by project or date.

## Tools

- `start_timer` — Start a timer for a task/project.
- `stop_timer` — Stop the active timer and record the session.
- `get_summary` — Summarize tracked time by project or date.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m time-tracker-ai-mcp.server
```
